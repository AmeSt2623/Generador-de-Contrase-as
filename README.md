# Generador-de-Contrase-as
Este programa es un generador de contraseñas seguras que utiliza ventanas emergentes (JOptionPane) para interactuar con el usuario. El sistema solicita la longitud deseada para la contraseña, valida que el dato sea correcto y luego crea una contraseña aleatoria usando letras mayúsculas, minúsculas, números y símbolos.
